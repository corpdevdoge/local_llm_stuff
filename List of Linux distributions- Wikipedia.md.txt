This page provides general information about notable [Linux distributions](/wiki/Linux_distribution "Linux distribution") in the form of a categorized list. Distributions are organized into sections by the major distribution or [package management system](/wiki/Package_management_system "Package management system") they are based on.

Debian-based
------------

\[[edit](/w/index.php?title=List_of_Linux_distributions&action=edit&section=1 "Edit section: Debian-based")\]

[![](//upload.wikimedia.org/wikipedia/commons/thumb/6/69/DebianFamilyTree1210.svg/170px-DebianFamilyTree1210.svg.png)](/wiki/File:DebianFamilyTree1210.svg)

Debian family tree

[Debian](/wiki/Debian "Debian") (a [portmanteau](/wiki/Portmanteau "Portmanteau") of the names "Deb" and "Ian") Linux is a distribution that emphasizes [free software](/wiki/Free_software "Free software"). It supports many hardware platforms. Debian and distributions based on it use the [.deb](/wiki/Deb_(file_format) "Deb (file format)") package format[\[2\]](#cite_note-2) and the [dpkg](/wiki/Dpkg "Dpkg") package manager and its frontends (such as [apt](/wiki/APT_(software) "APT (software)") or synaptic).[\[3\]](#cite_note-3)

Distribution

Description

[antiX](/wiki/AntiX "AntiX")

A comparatively lightweight distribution suitable for older computers, while also providing [kernel](/wiki/Linux_kernel "Linux kernel") and applications, as well as updates and additions via the [Aptitude](/wiki/Aptitude_(software) "Aptitude (software)") and Debian-compatible [repositories](/wiki/Repo "Repo").

[Astra Linux](/wiki/Astra_Linux "Astra Linux")

A Russian distribution developed to meet the needs of the Russian army and other armed forces and intelligence agencies.[\[4\]](#cite_note-4) It provides data protection up to the level of "top secret" in Russian classified information grade. It has been officially certified by Russian Defense Ministry, Federal Service for Technical and Export Control[\[5\]](#cite_note-5) and Federal Security Service.[\[6\]](#cite_note-6)

[Bharat Operating System Solutions](/wiki/Bharat_Operating_System_Solutions "Bharat Operating System Solutions") (BOSS)

An Indian Linux distribution[\[7\]](#cite_note-7)

[Canaima](/wiki/Canaima_(operating_system) "Canaima (operating system)")

A Venezuelan Linux distribution.[\[8\]](#cite_note-8)

[Corel Linux](/wiki/Corel_Linux "Corel Linux")

Short-lived commercial desktop Linux distribution, bought by [Xandros](/wiki/Xandros "Xandros") Linux.[\[9\]](#cite_note-9)

[CrunchBang Linux](/wiki/CrunchBang_Linux "CrunchBang Linux")

A small distribution and [Live CD](/wiki/Live_CD "Live CD") based on Debian Stable, featuring the [Openbox](/wiki/Openbox "Openbox") window manager and tint2 panel with [GTK+](/wiki/GTK%2B "GTK+") applications.[\[10\]](#cite_note-10) Development has ended for CrunchBang as of February, 2015.[\[11\]](#cite_note-11)

[Deepin](/wiki/Deepin "Deepin")

A Chinese Linux Distribution developed by Wuhan Deepin Technology Co.

[Devuan](/wiki/Devuan "Devuan")

A fork of Debian begun in 2014 with the primary goal of allowing user choice in [init](/wiki/Init "Init") systems, by decoupling software packages from [systemd](/wiki/Systemd "Systemd").[\[12\]](#cite_note-12)

[DoudouLinux](/wiki/DoudouLinux "DoudouLinux")

A discontinued distribution intended for children.

[Dreamlinux](/wiki/Dreamlinux "Dreamlinux")

A discontinued Brazilian distribution.

[Emdebian Grip](/wiki/Emdebian_Grip "Emdebian Grip")

A small-footprint distribution based on and compatible with Debian, intended for use on resource-limited embedded systems.[\[13\]](#cite_note-13)

[Finnix](/wiki/Finnix "Finnix")

A small system-administration [Live CD](/wiki/Live_CD "Live CD") that is available for multiple architectures[\[14\]](#cite_note-14)

[gLinux](/wiki/GLinux "GLinux")

gLinux is a [Linux Distro](/wiki/Linux_distribution "Linux distribution") used for [Google](/wiki/Google "Google") Employees.

[gNewSense](/wiki/GNewSense "GNewSense")

Originally based on [Ubuntu](/wiki/Ubuntu "Ubuntu") and later upon [Debian](/wiki/Debian "Debian"), and developed with sponsorship from the [Free Software Foundation](/wiki/Free_Software_Foundation "Free Software Foundation"). Its goal is user-friendliness, but with all [proprietary](/wiki/Proprietary_software "Proprietary software") (e.g. [binary blobs](/wiki/Binary_blob "Binary blob")) and non-free software removed.

[grml](/wiki/Grml "Grml")

Live CD for system recovery[\[15\]](#cite_note-15)

[HandyLinux](/wiki/HandyLinux "HandyLinux")

Designed for [senior citizens](/wiki/Senior_citizen "Senior citizen") running old computers for which [Windows](/wiki/Windows "Windows") have become too slow[\[16\]](#cite_note-16)

[Kali Linux](/wiki/Kali_Linux "Kali Linux")

A completely customizable distribution used for penetration testing. Kali is based on Debian and is used mostly by security experts.[\[17\]](#cite_note-17) Originally named [BackTrack](/wiki/BackTrack "BackTrack") (named after the homonym class of backtracking algorithms), it is developed by [Offensive Security](/wiki/Offensive_Security "Offensive Security").[\[18\]](#cite_note-BackTrack_website-18) In March 2013, the Offensive Security team rebuilt BackTrack on Debian and released it under the name Kali Linux.[\[19\]](#cite_note-19)

[Kali NetHunter](/wiki/Kali_NetHunter "Kali NetHunter")

Mobile version of Kali Linux available for rooted and non-rooted devices.

[Kanotix](/wiki/Kanotix "Kanotix")

An installable live DVD/CD for desktop usage using KDE and LXDE, focusing on convenient scripts and [GUIs](/wiki/GUI "GUI") for ease of use.[\[20\]](#cite_note-20)

[LEAF Project](/wiki/LEAF_Project "LEAF Project")

The Linux Embedded Appliance Framework. A tiny primarily floppy-based distribution for routers, firewalls and other appliances.[\[21\]](#cite_note-21)

[Libranet](/wiki/Libranet "Libranet")

A discontinued operating system based on Debian.

[LiMux](/wiki/LiMux "LiMux")

An [ISO 9241](/wiki/ISO_9241 "ISO 9241") industry workplace certified Linux distribution, deployed at the [City of Munich](/wiki/Munich "Munich"), Germany.[\[22\]](#cite_note-22)

[LMDE](/wiki/Linux_Mint_Debian_Edition "Linux Mint Debian Edition")

A Debian-based version of [Linux Mint](/wiki/Linux_Mint "Linux Mint") that does not use any elements of [Ubuntu linux](/wiki/Ubuntu_linux "Ubuntu linux"), maintained to ensure continuity should Ubuntu stop being maintained or other issue affecting the core Mint distribution.[\[23\]](#cite_note-23)

[Maemo](/wiki/Maemo "Maemo")

A development platform for hand held devices such as the [Nokia N800](/wiki/Nokia_N800 "Nokia N800"), [N810](/wiki/Nokia_N810 "Nokia N810"), and [Nokia N900](/wiki/Nokia_N900 "Nokia N900") Internet Tablets and other Linux kernel–based devices.[\[24\]](#cite_note-24)

[MEPIS](/wiki/MEPIS "MEPIS")

A discontinued OS that focused on ease of use. Significant derivatives include [antiX](/wiki/AntiX "AntiX") and [MX Linux](/wiki/MX_Linux "MX Linux").

[MintPPC](/wiki/MintPPC "MintPPC")

For PowerPC computers. Although MintPPC uses some Linux Mint Debian Edition code, it is not Linux Mint.[\[25\]](#cite_note-25)

[Musix GNU+Linux](/wiki/Musix_GNU%2BLinux "Musix GNU+Linux")

Intended for music production, graphic design, audio, video editing, and other tasks. It is built with only [free software](/wiki/Free_software "Free software").[\[26\]](#cite_note-26)

[MX Linux](/wiki/MX_Linux "MX Linux")

A midweight OS based on Debian Stable with core components from [antiX](/wiki/AntiX "AntiX") and using [Xfce](/wiki/Xfce "Xfce"), offering simple configuration, high stability, solid performance and medium-sized footprint.[\[27\]](#cite_note-27)

[NepaLinux](/wiki/NepaLinux "NepaLinux")

A Debian- and Morphix-based distribution focused for desktop usage in Nepali language computing.[\[28\]](#cite_note-28)

[OpenZaurus](/wiki/OpenZaurus "OpenZaurus")

Debian packages and ROM image for the [Sharp Zaurus](/wiki/Sharp_Zaurus "Sharp Zaurus") [PDA](/wiki/Personal_digital_assistant "Personal digital assistant"). Replaced by [Ångström distribution](/wiki/%C3%85ngstr%C3%B6m_distribution "Ångström distribution").[\[29\]](#cite_note-29)

[Pardus](/wiki/Pardus_(operating_system) "Pardus (operating system)")

Developed by [Turkish National Research Institute of Electronics and Cryptology](/wiki/Turkish_National_Research_Institute_of_Electronics_and_Cryptology "Turkish National Research Institute of Electronics and Cryptology"). Prior to 2013 it used PISI as the package manager, with COMAR as the configuration framework. Starting with Pardus 2013, it is Debian-based.

[Parrot OS](/wiki/Parrot_OS "Parrot OS")

A Linux distribution based on Debian used by penetration testers.

[Parsix](/wiki/Parsix "Parsix")[\[30\]](#cite_note-30)

Optimized for personal computers and laptops. Built on top of Debian testing branch and comes with security support.[\[31\]](#cite_note-31)

[PelicanHPC](/wiki/PelicanHPC "PelicanHPC")

Dedicated to setting up a [computer cluster](/wiki/Computer_cluster "Computer cluster").[\[32\]](#cite_note-32)

[PureOS](/wiki/PureOS "PureOS")

A Linux distribution based on Debian with a focus on privacy, security, and convenience.[\[33\]](#cite_note-33)[\[34\]](#cite_note-34)[\[35\]](#cite_note-35)[\[36\]](#cite_note-36)

[Q4OS](/wiki/Q4OS "Q4OS")

A [light-weight Linux distribution](/wiki/Light-weight_Linux_distribution "Light-weight Linux distribution") with [Trinity](/wiki/Trinity_Desktop_Environment "Trinity Desktop Environment") and [Plasma](/wiki/KDE_Plasma_Workspaces "KDE Plasma Workspaces") desktop environments.[\[37\]](#cite_note-37)

[Raspberry Pi OS](/wiki/Raspberry_Pi_OS "Raspberry Pi OS")

Desktop-oriented distribution, formerly known as Raspbian. Developed by the [Raspberry Pi Foundation](/wiki/Raspberry_Pi_Foundation "Raspberry Pi Foundation") as the official OS for their family of low-power [Raspberry Pi](/wiki/Raspberry_Pi "Raspberry Pi") single-board computers.

[SolydXK](/wiki/SolydXK "SolydXK")

[Xfce](/wiki/Xfce "Xfce") and [KDE](/wiki/KDE "KDE") desktop focused on stability, security and ease of use.[\[38\]](#cite_note-38)

[SparkyLinux](/wiki/SparkyLinux "SparkyLinux")

A Debian-based Linux distribution which provides ready to use, out of the box operating system with a set of slightly customized lightweight desktops. Sparky is targeted to all the computer's users who want replace existing, proprietary driven OS to an open-sourced..

[Sunwah Linux](/wiki/Sunwah_Linux "Sunwah Linux")

A Chinese Linux distribution[\[39\]](#cite_note-39)

[Tails](/wiki/Tails_(operating_system) "Tails (operating system)")

Tails or The Amnesic Incognito Live System' is aimed at preserving privacy and anonymity, with all outgoing connections forced to go through [Tor](/wiki/Tor_(anonymity_network) "Tor (anonymity network)").[\[40\]](#cite_note-40)

[TurnKey Linux](/wiki/TurnKey_Linux_Virtual_Appliance_Library "TurnKey Linux Virtual Appliance Library")

Open source project developing a family of free, Debian-based appliances optimized for ease of use in server-type usage scenarios.[\[41\]](#cite_note-41) Based on Debian since 2012; previously based on Ubuntu.

[Twister OS](/wiki/Twister_OS "Twister OS")

[Raspberry Pi OS](/wiki/Raspberry_Pi_OS "Raspberry Pi OS") based distribution using [Xfce](/wiki/Xfce "Xfce") with themes based on other OSes intended for the [Raspberry Pi](/wiki/Raspberry_Pi "Raspberry Pi"), [RK3399](/wiki/Rockchip#Featured_Products "Rockchip") CPU, and [x86-64](/wiki/X86-64 "X86-64") architecture.

[Univention Corporate Server](/wiki/Univention_Corporate_Server "Univention Corporate Server")

Enterprise distribution with integrated IT infrastructure and identity management system by the company Univention GmbH, Germany. A full version for up to 5 users for tests and for private use can be downloaded for free.[\[42\]](#cite_note-42)

[Webconverger](/wiki/Webconverger "Webconverger")

[Debian Live](/wiki/Debian_Live "Debian Live")\-based browser only distribution, similar to ChromeOS. However based on [Firefox](/wiki/Firefox "Firefox") and [dwm](/wiki/Dwm "Dwm"), with no user sign-in, no special hardware required and designed for public places.[\[43\]](#cite_note-43)

[Vyatta](/wiki/Vyatta "Vyatta")

Commercial open source network operating system includes routing, firewall, VPN, intrusion prevention and more. Designed to be an open source Cisco replacement.[\[44\]](#cite_note-44)

[VyOS](/wiki/VyOS "VyOS")

Free routing platform. Because VyOS is run on standard amd64, i586 and ARM systems, it is able to be used as a router and firewall platform for cloud deployments.

### Ubuntu-based

\[[edit](/w/index.php?title=List_of_Linux_distributions&action=edit&section=2 "Edit section: Ubuntu-based")\]

[![](//upload.wikimedia.org/wikipedia/commons/thumb/7/79/UbuntuFamilyTree1210.svg/170px-UbuntuFamilyTree1210.svg.png)](/wiki/File:UbuntuFamilyTree1210.svg)

Ubuntu family tree

[Ubuntu](/wiki/Ubuntu "Ubuntu") (named after the [Nguni](/wiki/Nguni_people "Nguni people") philosophy of _[ubuntu](/wiki/Ubuntu_philosophy "Ubuntu philosophy")_) is a distribution based on Debian, designed to have regular releases, a consistent user experience and commercial support on both desktops and servers.

#### Current official derivatives

\[[edit](/w/index.php?title=List_of_Linux_distributions&action=edit&section=3 "Edit section: Current official derivatives")\]

These Ubuntu variants, also known as Ubuntu flavours, simply install a set of packages different from the original Ubuntu, but since they draw additional packages and updates from the same repositories as Ubuntu, all of the same software is available for each of them.[\[45\]](#cite_note-45)[\[46\]](#cite_note-46)

Distribution

Description

[Edubuntu](/wiki/Edubuntu "Edubuntu")

A complete Linux based operating system that was targeted for [primary](/wiki/Primary_education "Primary education") and [secondary education](/wiki/Secondary_education "Secondary education"). Outdated versions are freely available with community-based support. The Edubuntu community is built on the ideas enshrined in the Edubuntu Manifesto: that software, especially for education, should be available [free of charge](/wiki/Gratis_versus_Libre "Gratis versus Libre") and that software tools should be usable by people in their local language and despite any disabilities.[\[47\]](#cite_note-47)

[Kubuntu](/wiki/Kubuntu "Kubuntu")

An official derivative of Ubuntu Linux using [KDE](/wiki/KDE "KDE") instead of the [GNOME](/wiki/GNOME "GNOME") (or Unity) desktop environment used by default in Ubuntu.[\[48\]](#cite_note-48)

[Lubuntu](/wiki/Lubuntu "Lubuntu")

An official derivative of the [Ubuntu](/wiki/Ubuntu "Ubuntu") [operating system](/wiki/Operating_system "Operating system") that is "lighter, less resource hungry and more energy-efficient", using the [LXQt](/wiki/LXQt "LXQt") [desktop environment](/wiki/Desktop_environment "Desktop environment") (used [LXDE](/wiki/LXDE "LXDE") before 18.10).[\[49\]](#cite_note-Distrowatch11May09-49)[\[50\]](#cite_note-LXDEBlog-50)[\[51\]](#cite_note-51)

[Ubuntu Budgie](/wiki/Ubuntu_Budgie "Ubuntu Budgie")

An official derivative of Ubuntu using [Budgie](/wiki/Budgie_(desktop_environment) "Budgie (desktop environment)").

[Ubuntu Cinnamon](/wiki/Ubuntu_Cinnamon "Ubuntu Cinnamon")

An official derivative of Ubuntu using the [Cinnamon](/wiki/Cinnamon_(desktop_environment) "Cinnamon (desktop environment)") desktop environment.[\[52\]](#cite_note-52)

[Ubuntu Kylin](/wiki/Ubuntu_Kylin "Ubuntu Kylin")

An official derivative aimed at the Chinese market.

[Ubuntu MATE](/wiki/Ubuntu_MATE "Ubuntu MATE")

An official derivative of Ubuntu using [MATE](/wiki/MATE_(software) "MATE (software)"), a desktop environment forked from the now-defunct GNOME 2 code base, with an emphasis on the [desktop metaphor](/wiki/Desktop_metaphor "Desktop metaphor").[\[53\]](#cite_note-Sneddon01Mar15-53)

[Ubuntu Server](/wiki/Ubuntu_Server "Ubuntu Server")

An official derivative made for use in servers & [IBM mainframes](/wiki/IBM_mainframe "IBM mainframe"). Ubuntu Server handles mail, controls printers, acts as a fileserver, can host [LAMP](/wiki/LAMP_(software_bundle) "LAMP (software bundle)") and more.[\[54\]](#cite_note-54)

[Ubuntu Studio](/wiki/Ubuntu_Studio "Ubuntu Studio")

Based on Ubuntu, providing open-source applications for [multimedia](/wiki/Multimedia "Multimedia") creation aimed at the audio, video and graphic editors.[\[55\]](#cite_note-55)

[Ubuntu Unity](/wiki/Ubuntu_Unity "Ubuntu Unity")

An official derivative of Ubuntu using the [Unity](/wiki/Unity_(user_interface) "Unity (user interface)") desktop environment.[\[56\]](#cite_note-Evangelho12May20-56)[\[57\]](#cite_note-57)

[Xubuntu](/wiki/Xubuntu "Xubuntu")

An official derivative of Ubuntu using [Xfce](/wiki/Xfce "Xfce"). Xubuntu is intended for use on less-powerful computers or those who seek a highly efficient desktop environment on faster systems, and uses mostly [GTK](/wiki/GTK "GTK") applications.[\[58\]](#cite_note-58)

#### Discontinued official derivatives

\[[edit](/w/index.php?title=List_of_Linux_distributions&action=edit&section=4 "Edit section: Discontinued official derivatives")\]

Distribution

Description

[Gobuntu](/wiki/Gobuntu "Gobuntu")

Gobuntu was an official derivative of the Ubuntu operating system, aiming to provide a distribution consisting entirely of [free software](/wiki/Free_software "Free software"). It was officially announced by Mark Shuttleworth on July 10, 2007, and daily builds of Gobuntu 7.10 began to be publicly released. The project ended around the release of 8.04 and has since merged into mainline Ubuntu as a 'free software' option.[\[59\]](#cite_note-59)

[Mythbuntu](/wiki/Mythbuntu "Mythbuntu")

Based on Ubuntu and [MythTV](/wiki/MythTV "MythTV"), providing applications for recording TV and acting as a media center.[\[60\]](#cite_note-60) On 4 November 2016 the development team announced the end of Mythbuntu as a separate distribution, citing insufficient developers.

[Ubuntu GNOME](/wiki/Ubuntu_GNOME "Ubuntu GNOME")

Formerly an official Ubuntu variant,[\[61\]](#cite_note-61) but since the main Ubuntu 17.10, which uses [GNOME Shell](/wiki/GNOME_Shell "GNOME Shell") as its default desktop and GDM as its display manager, this distro has been merged into mainline releases.[\[62\]](#cite_note-62)

[Ubuntu JeOS](/wiki/Ubuntu_JeOS "Ubuntu JeOS")

"Just Enough OS" – was described as "an efficient variant \[...\] configured specifically for [virtual appliances](/wiki/Virtual_appliance "Virtual appliance")".[\[63\]](#cite_note-63) Since the release of Ubuntu 8.10 it has been included as an option as part of the standard Ubuntu Server Edition.

[Ubuntu Mobile](/wiki/Ubuntu_Mobile "Ubuntu Mobile")

An embedded operating system designed for use on [mobile devices](/wiki/Mobile_device "Mobile device"). The operating system will use [Hildon](/wiki/Hildon "Hildon") from [maemo](/wiki/Maemo "Maemo") as its [graphical frontend](/wiki/Graphical_user_interface "Graphical user interface"). [Ubuntu Touch](/wiki/Ubuntu_Touch "Ubuntu Touch") is a successor to Ubuntu Mobile.

[Ubuntu Netbook Edition](/wiki/Ubuntu_Netbook_Edition "Ubuntu Netbook Edition")

Netbook Edition was an official derivative of Ubuntu designed for [netbooks](/wiki/Netbook "Netbook") using the [Intel Atom](/wiki/Intel_Atom "Intel Atom") processor. Starting from [Ubuntu 11.04](/wiki/Ubuntu_version_history#1104 "Ubuntu version history"), Ubuntu Netbook Edition has been merged into the desktop edition.[\[64\]](#cite_note-64)

[Ubuntu TV](/wiki/Ubuntu_TV "Ubuntu TV")

Designed for use with [TVs](/wiki/Smart_TV "Smart TV").[\[65\]](#cite_note-65)

#### Unofficial derivatives

\[[edit](/w/index.php?title=List_of_Linux_distributions&action=edit&section=5 "Edit section: Unofficial derivatives")\]

Unofficial variants and derivatives are not controlled or guided by [Canonical Ltd.](/wiki/Canonical_Ltd. "Canonical Ltd.") and generally have different goals in mind.

Distribution

Description

[BackBox](/wiki/BackBox "BackBox")

BackBox is a Linux distribution based on Ubuntu. It has been developed to perform penetration tests and security assessments. Designed to be fast, easy to use and provide a minimal yet complete desktop environment, thanks to its own software repositories, always being updated to the latest stable version of the most used and best known ethical hacking tools.[\[66\]](#cite_note-66)

[Bodhi Linux](/wiki/Bodhi_Linux "Bodhi Linux")

An Ubuntu-based Linux distribution featuring the Moksha Desktop environment and targeting users who want a minimum of preinstalled software or low system requirements.[\[67\]](#cite_note-67)

[Cub Linux](/wiki/Cub_Linux "Cub Linux")

Ubuntu-based distribution designed to mimic the desktop appearance and functionality of ChromeOS.[\[68\]](#cite_note-68)[\[69\]](#cite_note-69)[\[70\]](#cite_note-70)

[dyne:bolic](/wiki/Dyne:bolic "Dyne:bolic")

[Live CD](/wiki/Live_CD "Live CD") geared toward multimedia (audio and video) production, but comes with other non-media specific application (e.g. word processor, desktop publisher)[\[71\]](#cite_note-71)

[EasyPeasy](/wiki/EasyPeasy "EasyPeasy")

Fork of Ubuntu designed for netbooks[\[72\]](#cite_note-72)

[Eeebuntu](/wiki/Eeebuntu_OS "Eeebuntu OS")

Specifically for the Eee PC range of netbooks, based on Debian. Also rebranded as Aurora OS.[\[73\]](#cite_note-73)

[Element OS](/wiki/Element_OS "Element OS")

Based on [Xubuntu](/wiki/Xubuntu "Xubuntu"), made for [Home theater PCs](/wiki/Home_theater_PC "Home theater PC")[\[74\]](#cite_note-74)

[Elive](/wiki/Elive "Elive")

A [light-weight Linux distribution](/wiki/Light-weight_Linux_distribution "Light-weight Linux distribution") featuring the [Enlightenment](/wiki/Enlightenment_(software) "Enlightenment (software)") desktop, designed to be simple and suitable for new Linux users who want a fully functional environment.

[elementary OS](/wiki/Elementary_(operating_system) "Elementary (operating system)")

A distribution focusing mainly on non-technical users, has a [pay what you want](/wiki/Pay_what_you_want "Pay what you want") model.[\[75\]](#cite_note-75)

[Emmabuntüs](/wiki/Emmabunt%C3%BCs "Emmabuntüs")

[Arch Linux](/wiki/Arch_Linux "Arch Linux") is an independently developed, x86-64 general-purpose Linux distribution that strives to provide the latest stable versions of most software by following a rolling-release model. ### Mandriva-based

\[[edit](/w/index.php?title=List_of_Linux_distributions&action=edit&section=14 "Edit section: Mandriva-based")\]

[Mandriva Linux](/wiki/Mandriva_Linux "Mandriva Linux") is open-source distribution (with exceptions), discontinued in 2011. The first release was named Mandrake Linux and based on Red Hat Linux (version 5.1) and [KDE](/wiki/KDE "KDE") 1 in July 1998. It had since moved away from Red Hat's distribution and became a completely separate distribution. The name was changed to Mandriva, which included a number of original tools, mostly to ease system configuration. Mandriva Linux was the brainchild of [Gaël Duval](/wiki/Ga%C3%ABl_Duval "Gaël Duval"), who wanted to focus on ease of use for new users. The last stable version was in 2011. Mandriva's developers moved to [Mageia](/wiki/Mageia "Mageia")[\[114\]](#cite_note-114) and [OpenMandriva](/wiki/OpenMandriva "OpenMandriva").[\[115\]](#cite_note-115)

[CRUX](/wiki/CRUX "CRUX")

CRUX is a lightweight, [x86-64](/wiki/X86-64 "X86-64")\-optimized Linux distribution targeted at experienced users. The focus is "[keep it simple](/wiki/KISS_principle "KISS principle")", which is reflected in a simple [tar.xz](/wiki/Tar.xz "Tar.xz")\-based package system, [BSD](/wiki/BSD "BSD")\-style initscripts, and a relatively small collection of trimmed packages. [GoboLinux](/wiki/GoboLinux "GoboLinux")

A Linux distribution which redefines the [file system](/wiki/File_system "File system") hierarchy by installing everything belonging to one application in one folder under /Programs, and using symlinks from /System and its subfolders to point to the proper files.

[NixOS](/wiki/NixOS "NixOS")
